1. What is CSS?

CSS is a stylesheet language that controls the presentation of HTML documents. It defines how elements look: colors, fonts, spacing, layout, animations, responsiveness, and more.

  • It separates content (HTML) from design (CSS).

2. CSS Syntax

selector {
  property: value;
}
  • Selector: Targets an element (p, .class, #id).
  • Property: The aspect you want to style (color, font-size, margin).
  • Value: The setting (red, 16px, 10px).
h1 {
  color: blue;
  font-size: 24px;
}

3. Ways to Apply CSS

There are three main ways to add CSS:

  • Inline CSS ‚Äì inside an element.
  • Internal CSS ‚Äì inside a <style> in <head>.
  • External CSS ‚Äì in a separate .css file (recommended).
<p style="color: red;">Hello</p>

<style>
  p { color: red; }
</style>

<link rel="stylesheet" href="style.css">
üëâ External CSS is preferred for clean, reusable code.

4. Selectors

Selectors determine which elements CSS rules apply to.

  • Basic Selectors: p, #id, .class, *
  • Combinators: div p, div > p, div + p, div ~ p
  • Attribute Selectors: [attr], [attr="value"], [attr^="value"]
  • Pseudo-classes: :hover, :active, :first-child, :nth-child(2n)
  • Pseudo-elements: ::before, ::after, ::first-line

5. CSS Units

  • Absolute: px, pt, cm, mm, in
  • Relative: %, em, rem, vw, vh, vmin, vmax

6. Colors

  • Named: red, blue
  • HEX: #ff0000
  • RGB: rgb(255,0,0)
  • RGBA: rgba(255,0,0,0.5)
  • HSL: hsl(0,100%,50%)

8. The Box Model

Every element is a rectangular box with layers:

  • Content ‚Üí text or image
  • Padding ‚Üí space inside border
  • Border ‚Üí thickness around padding
  • Margin ‚Üí space outside border
div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 30px;
  box-sizing: border-box;
}

Position: static, relative, absolute, fixed, sticky

19. Responsive Design

@media (max-width: 768px) {
  body { font-size: 14px; }
}

20. CSS Variables

:root {
  --main-color: #3498db;
  --padding: 10px;
}
div {
  color: var(--main-color);
  padding: var(--padding);
}

21. CSS Functions

width: calc(100% - 50px);
font-size: clamp(14px, 2vw, 18px);
color: rgba(255,0,0,0.7);

22. Advanced Features

  • Filters: blur, brightness, contrast, grayscale
  • Blend Modes: mix-blend-mode
  • Clip Path: clip-path: circle(50%);
  • Masking and Shape Outside
  • CSS Subgrid (Grid Level 2)

23. CSS Best Practices

  • Keep CSS modular (use external files).
  • Use meaningful class names (e.g., BEM: .block__element--modifier).
  • Avoid too many !important rules.
  • Use variables for consistency.
  • Minify CSS for production.
  • Consider frameworks (Bootstrap, Tailwind, Bulma).
  • Test across browsers.
