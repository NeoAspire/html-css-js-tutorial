1. What is JavaScript?

JavaScript is a high-level, dynamic, interpreted programming language.

  • Originally built for browsers to make web pages interactive.
  • Now runs everywhere: frontend, backend, mobile apps, desktop apps, IoT.
  • Follows the ECMAScript (ES) standard.
  • Prototype-based OOP, weakly typed, dynamic, single-threaded with an event loop.

15. Asynchronous JavaScript

// Callback
setTimeout(() => console.log("Done"), 1000);

// Promise
let promise = new Promise((resolve,reject)=>{
  setTimeout(()=>resolve("Success"),1000);
});
promise.then(res=>console.log(res));

// Async/Await
async function fetchData() {
  try {
    let res = await fetch("https://api.example.com");
    let data = await res.json();
    console.log(data);
  } catch(err) {
    console.error(err);
  }
}
