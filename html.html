Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to structure web pages.

  • HTML uses elements (tags) to mark up content.
  • Browsers read HTML and display it for users.
  • HTML is not a programming language â€” itâ€™s a markup language.

Basic Structure of a Web Page

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>
